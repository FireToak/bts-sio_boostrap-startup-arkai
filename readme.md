# Site vitrine de la StartUP ArkAI

## Contexte

Ce projet a été réalisé dans le cadre de ma formation en **BTS SIO** (Services Informatiques aux Organisations). L'objectif était de concevoir un site vitrine responsive en utilisant le framework **Bootstrap**, dans le cadre des cours de **référencement web (SEO)**. Le but est de structurer un site performant et optimisé pour présenter une activité professionnelle.

## Présentation de ma StartUP Fictive

**ArkAI** est une startup *Green by design* fondée par Louis MEDO, proposant une solution d'intelligence artificielle vertueuse nommée **Feronia**.

L'objectif d'ArkAI est de réduire l'impact carbone exponentiel de l'IA générative grâce à une innovation technologique majeure : le **Switch Learning**.
* **Concept :** Utilisation des unités asynchrones des CPU standards pour maximiser l'efficience matérielle.
* **Avantages :** Supprime la dépendance aux GPU énergivores (type H100) et permet une réduction estimée de **40% de la consommation d'énergie**.
* **Slogan :** « Une intelligence au service de l'écologie. »
